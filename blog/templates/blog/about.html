About BlogVerse

BlogVerse is a simple blogging platform built with Django. It allows users to create, read, update, and delete blog posts easily. The application features user authentication, enabling users to register, log in, and manage their own posts.

Key features of BlogVerse include:

  • User registration and authentication
  • Create and manage blog posts
  • Responsive design using Bootstrap
  • Image uploads for blog posts

Technologies used in this project:

  • Django
  • HTML/CSS
  • JavaScript
  • SQLite (for development)

We hope you enjoy using BlogVerse!
